
There is a setup screen that allows the user to select options for filling.
The password is 1080.  This can be changed.

Setup Options:
--------------
Target Weight - What is the target weight you are trying to hit.
PreAct        - How soon before target do you want filling shut off.
Band          - This is an acceptance band.  If the load falls short of target by this amount
        	then the user is presented with a Bump button.  Pressing this will fill the
        	scale while it is pressed.  Allowing the user to get closer to target.

Operation:
----------
User presses the Select Button to select which bay will be filled.

The program looks for the divert switch to close for that bay.

After the divert switch is detected the scale will wait for up to 60 seconds for Start to be pressed
 - if start is not pressed within 60 seconds the program resets.

User presses 'Start' to begin filling.
 - The scale looks at the current weight on the scale and records it.  It will adjust your
   filling operation to account for this weight.

User can press 'Stop' to pause filling.

If the user presses 'Start' again the scale will begin filling again.

To Cancel Filling you need to press the 'Cancel' on the 920

System will fill until the weight on the scale is equal to or greater than
 Your target weight - the PreAct weight.

The 920 will look for the scale to steady on a weight and see if the weight is within the limits.

 - If the FallOff is within limits the system will automatically save the load and reset.

 - If it is short the display will read "Short" and Bump and Finish buttons will be present on the 920
 - Bump will dispense product while it is depressed.
 - If the scale steadies out within acceptable limits the scale will automatically save the weight and reset.
 - If the user presses "Finish" on the 920 it will save the current tote and reset.
 - If the user presses a Select button while the scale is short, the scale will save the current tote and reset.
